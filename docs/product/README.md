# Product Docs

Product docs translate research conclusions into product decisions.

This directory is not for API specs, database schemas, screen-level UI specs, or product code.

## Contents

- `product-shape.md` — long-term product shape and philosophy
- `first-launch-product-design.md` — first launch product design, scoped to launch and revenue readiness

## Rule

Do not turn abstract capabilities into product features directly. Use this chain:

```text
Long-term capability
-> Product object
-> User action
-> System feedback
-> Validation signal
-> First launch inclusion decision
```
