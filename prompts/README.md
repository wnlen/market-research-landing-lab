# Prompts

跨 Agent 协作时使用的提示词。调研流程中反复使用、效果稳定的提示词才放这里；一次性指令不留档。

## 建议子目录

- `claude/` — 给 Claude Code 的任务提示词
- `codex/` — 给 Codex 的任务提示词
- `shared/` — 可复用的提示词片段（输出规范、三段式要求、边界约束）

## 规则

- 每个提示词文件顶部写明：用途、输入、输出、适配的 Agent
- 与 `skills/` 区分：skill 是稳定工作流的合约；prompt 是即用即走的指令
- 提示词里不内嵌具体案例数据；数据留在 `research/` 或 `docs/`
