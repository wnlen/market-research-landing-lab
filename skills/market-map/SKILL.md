# market-map

## 用途

把一个候选赛道整理为可比较的市场地图模板。

## 输入

- 赛道名称
- 已有原始资料
- 已有竞品卡片

## 输出

- 更新 `docs/01-market-map.md`
- 必要时补充 `research/source-cards/` 或 `research/competitor-cards/`

## 必须遵守

- 不新增未经验证的市场事实
- 原始材料和结构化判断分开
- 所有结论使用 `证据 / 推断 / 决策`

## 推荐步骤

1. 先确认本轮只研究一个赛道
2. 汇总已有材料
3. 找出市场结构、用户、商业模式占位
4. 标记证据缺口
5. 给出是否继续研究该赛道的决策建议

## 完成标准

- 文档结构完整
- 证据链可追溯
- 没有把猜测写成结论
