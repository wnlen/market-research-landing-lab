# handoff

## 用途

在 Claude Code、Codex、ChatGPT 之间做高质量交接，减少重复劳动和信息漂移。

## 输入

- 当前阶段
- 已更新文件
- 已形成判断
- 证据缺口

## 输出

- 更新 `ai/handoff-to-codex.md` 或 `ai/handoff-to-claude.md`
- 必要时同步 `ai/state.md` 和 `ai/result.md`

## 交接最少要素

- 目标是什么
- 输入材料是什么
- 哪些结论已成立
- 哪些只是推断
- 下一步必须避免什么

## 必须遵守

- 不省略关键约束
- 不把假设伪装成共识
- 不要求对方从零重建上下文

## 完成标准

- 接手方可直接开工
- 风险和边界写清楚
